Three.js 入门学习日记

发布于 2026-05-07 技术  Three.js WebGL 3D图形 前端 

初识 Three.js

今天开始学习 Three.js，这是一个基于 WebGL 的 3D 库，可以让我们在浏览器中创建炫酷的 3D 效果。之前一直对 3D 编程感到神秘，终于迈出了第一步！

基础概念

在开始写代码之前，先理清了几个核心概念：

  1. Scene（场景） - 相当于一个容器，所有的物体、灯光、相机都要放在场景里
  2. Camera（相机） - 决定了我们从哪个角度看场景，常用的是 PerspectiveCamera（透视相机）
  3. Renderer（渲染器） - 负责把场景和相机渲染成实际的画面
  4. Mesh（网格） - 由 Geometry（几何体）和 Material（材质）组成，是我们看到的实际物体

遇到的坑

  1. 相机位置 - 一开始相机和立方体在同一个位置，什么都看不到，后来才记得要把相机往后移
  2. MeshBasicMaterial - 这个材质不响应光照，换了 MeshLambertMaterial 后灯光才有效果
  3. 渲染循环 - 必须用 requestAnimationFrame，不能用 setInterval，性能差很多

下一步计划

  • 学习灯光和阴影的使用
  • 尝试加载外部 3D 模型
  • 研究几何体顶点操作
  • 学习粒子系统

总的来说，Three.js 的 API 设计得很友好，入门比想象中简单。接下来要深入学习，希望能做出一些有趣的 3D 交互效果！
